Activity One

In the first activity, we discussed IoT applications and provided team-based responses to a few questions. Activity 2's discussion of smart buildings required us to first define the term "smart," which basically meant "intelligent networking and automation, energy consumption, and independent activities." Afterwards, we addressed the critical features that were required, some of which included automation, multifunctionality, adaptability, security, and real-time Next, we talked about some of the scenarios for smart buildings.
